# Quick Start Guide

## Initial Setup

1. **Install CLI dependencies:**
   ```bash
   cd cli-workflow
   npm install
   ```

2. **Create `.env` file:**
   ```bash
   # Copy the example (if it exists) or create manually
   echo "GEMINI_API_KEY=your_api_key_here" > .env
   ```

   Get your API key from: https://makersuite.google.com/app/apikey

3. **Add URLs to scrape:**
   Edit `cli-workflow/scrapeAndGenerate.js` and add URLs to the `inputURLs` array.

4. **Run the scraper:**
   ```bash
   cd cli-workflow
   npm run scrape
   ```

5. **Copy assets to main project:**
   ```bash
   # From project root
   npm run copy:assets
   ```

## Using the New Inspiration Page

The new `InspirationPage.tsx` component loads content from Markdown files generated by the CLI.

**Note:** The existing `Inspiration.tsx` page is still active in the router. To use the new filtered page:

1. Option A: Replace the import in `src/router/AppRouter.tsx`:
   ```typescript
   const Inspiration = React.lazy(() => import('../pages/InspirationPage'));
   ```

2. Option B: Add a new route (e.g., `/inspiration-filtered`) for the new page

3. Option C: Integrate the filtering functionality into the existing Inspiration page

## Build Integration

The build process automatically:
- Generates content (`npm run generate:content`)
- Copies images (`npm run copy:assets`)
- Cleans types
- Builds the React app

Run `npm run build` to execute the full workflow.
